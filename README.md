# NFC-uid-ReaderApp
App to Read the uids from NFC cards.  
I Use the ACR122U NFC Reader so i don't know if the app is compatible with other reader.  
The CSV file is here : NFC-uid-ReaderApp\importNfcUid\importNfcUid\bin\Debug\net6.0\uids.csv
