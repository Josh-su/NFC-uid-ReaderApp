# NFC-uid-ReaderApp
App to Read the uids from NFC cards and store them in a csv file in the format: card name, card uid.  
The CSV file is here : NFC-uid-ReaderApp\importNfcUid\importNfcUid\bin\Debug\net6.0\uids.csv  
I Use the ACR122U NFC Reader so i don't know if the app is compatible with other reader.  
You have to install the driver before launche the app https://www.acs.com.hk/download-driver-unified/9840/ACS-Unified-MSI-4280.rar
